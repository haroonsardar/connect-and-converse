# Connect & Converse - University Discussion Hub

A web-based discussion portal built as part of an FYP/Project.

### Features Implemented:
* **User Auth**: Registration, Login, and Password Recovery.
* **Admin Panel**: Management of Users, Posts, and Categories.
* **Discussions**: Thread creation, Commenting, and Voting (Upvote/Downvote).
* **Search**: Advanced search by keywords, categories, and date.

### Setup Instructions:
1. Import the `connect_converse_db.sql` file from the `/database` folder into your phpMyAdmin.
2. Update `config/db.php` with your local database credentials.
3. Open `localhost/connect-and-converse` in your browser.
